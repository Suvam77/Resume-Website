# Resume-Website
This is my Resume Repository. 
This repository contains the source code for my personal resume website. It is designed to present my academic background, professional experience, skills, projects, and contact information in a simple and structured way.

The website serves as an online version of my resume and can be easily updated as my profile evolves. It is suitable for academic, teaching, and industry roles.

Features

Clean and minimal resume-style layout

Sections for education, experience, skills, and projects

Easy to customize and maintain

Lightweight and fast to load

Technologies Used

HTML

CSS

JavaScript (if applicable)

Purpose

The goal of this project is to maintain a single, continually updated online resume that can be easily shared with recruiters, institutions, and collaborators.
